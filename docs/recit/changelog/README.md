## [1.0.1] - 2018.12

| 功能模块            | 版本号   |
| --------------- | ----- |
| RecIt-Scheduler | 1.0.1 |
| RecIt-Worker    | 1.0.1 |

### 变更

- 如果申请停止录制/推流时对应的资源尚在申请中，将返回成功而非失败。
- 录制和推流中，为音频增加了 1.5 秒时延，以改善音视频不同步的状况。

| RecIt-Worker    | 1.0.0 |

首个正式发布版本。

### 修正

- 如果申请停止录制/推流时对应的资源尚在申请中，停止请求也会被正确处理，对应的资源创建将被取消。
